Test 11: 8 WebGL contexts created simultaneously — no rendering

The main example page loads 7 JS modules, each calling new Plot(...) at module level (before any plot.update()). This test creates 8 Plot objects all at once and then calls plot.update() on them one at a time, fully awaiting each. If creating many contexts alone crashes, the bug is context creation. If sequential updates crash but test 07 (concurrent) does not, this test isolates the GPU budget per update.
