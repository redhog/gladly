Test 05: texImage2D with RGBA (not RGBA32F) internal format + FLOAT type

In WebGL2, the correct internal format for a 32-bit float RGBA texture is RGBA32F. Gladly's GlBase.js monkey-patches gl.texImage2D to auto-upgrade calls that use RGBA + FLOAT (the old WebGL1 way) to RGBA32F, implying the underlying code calls it incorrectly. This test calls texImage2D with the raw RGBA + FLOAT combination without the monkey-patch, to check whether the Nvidia driver crashes or just returns an error instead.

It also tests the correct RGBA32F form as a control.
