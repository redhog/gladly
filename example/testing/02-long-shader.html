Test 02: Single draw call with very long fragment shader

Renders a fullscreen quad where each fragment runs a 500,000-iteration loop. The loop has a data dependency (Mandelbrot-like iteration) so the GPU compiler cannot eliminate it. On an 800×600 canvas that is 480M × 500K = 240 billion dependent operations. A fast GPU may complete this in a few seconds; a slow one will trigger the Windows TDR (~2s GPU timeout).
